Определение термина "Tag"

Основное определение
Tag (англ. "метка") - элемент разметки в языках программирования и веб-разработке, используемый для структурирования и форматирования содержимого.
Характеристики
  1. Имеет открывающую и закрывающую части
  2. Может содержать атрибуты
  3. Определяет семантику содержимого

Примеры тегов

  1. - Заголовок первого уровня

  2. - Гиперссылка
